dConnectSDKSampleプロジェクトは Device Connect SDK for iOS ならびに各デバイスプラグインの評価用アプリです。

当プロジェクトには Device Connect SDK for iOS のframeworkファイル、及びリソースファイル
が組み込まれています。
各デバイスプラグインを評価する場合は、それぞれのbuildマニュアルに従い、各デバイスプラグインの
frameworkファイル、リソースファイル、及びその他必要なファイルの組み込み、設定を行いビルドしてください。

* 使用に関する留意事項

各デバイスに対して、任意のAPIを操作する画面では、リクエストの送信方法が二つ用意されています。

1. 「送信」ボタン押下によるリクエスト実行
2. 「イベント」ボタン押下によるリクエスト実行

1. は非イベントAPIを実行する場合に利用します。
APIのパス、パラメータを設定後押下することでリクエストを実行し
結果を受けると別画面に結果が表示されます。

2. はイベントAPIを実行する場合に利用します。
イベントの登録または解除のAPIのパス、パラメータを設定後押下することでリクエストを実行し
イベントメッセージを受けるとメッセージ内容が画面下部に表示されます。
